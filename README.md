# AURA Cloud Studio

AI-Powered Digital Audio Workstation (Hybrid Architecture).

## Running the App

1. Run `run_app.bat` to launch everything (Backend, Frontend, Electron).

## Development

- `npm run dev`: Start Web Server
- `npm run electron`: Start Electron App
- `npm run start:backend`: Start Python Engine

## Structure

- `src/renderer`: Frontend (React/Vite)
- `src/main`: Electron Main Process
- `src/python`: Python Backend (AI/Audio Engine)
